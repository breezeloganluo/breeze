<static> API.show(viewId, data, target)
显示视图，即在本APP中显示一个指定的视图，视图即在html中，声明于app下的任何顶级带id的标签,建议这些标签id为view_前缀 
this.API.show("view1",data);
Parameters:
{String} viewId
要显示的视图名称，某个顶级的标签的id
{Object} data
视图中使用的解析数据，在视图中将使用data引用这个对象
{String} target
视图显示的目标selector（jquery的selector），视图默认（即不填写该参数）将会把视图显示到app中，如果不是要覆盖当前视图，而是把这个视图显示到当前视图下都某个标签下，则填写该标签的selector
